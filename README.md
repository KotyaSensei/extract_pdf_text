# PDF Text Extractor

Простая утилита для извлечения текста из PDF-файлов с максимальным сжатием выходных данных.

## Особенности

- Автоматическая обработка всех `.pdf` файлов в текущей директории
- Минимизация объёма текста:
  - Каждая страница преобразуется в одну строку
  - Все переносы строк заменяются на пробелы
  - Лишние пробелы сжимаются в один
  - Пустые строки удаляются
- Результат сохраняется в `.txt` файлы с исходными именами
- Полная поддержка UTF-8 (кириллица, спецсимволы)

## Установка

```bash
pip install PyMuPDF
```

## Использование

1. Поместите `extract_pdf_text.py` в папку с PDF-файлами
2. Запустите скрипт:

```bash
python extract_pdf_text.py
```

3. Для каждого `файл.pdf` будет создан компактный `файл.txt`


## Требования

- Python 3.6+
- PyMuPDF (fitz)
